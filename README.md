### Farmcycle
an Automated Aquaponics

![IMG20230226162128](https://github.com/user-attachments/assets/07e8b903-a814-49da-9d66-1007fafb4802)

<h3> Pros: </h3>

- Easy monitoring of the temperature of the fish and chickens
- You can monitor the humidity and temperature of the room with the use of DHT22
- It has a fan which can be controlled through your phone for your chickens
- It has lights for lettuce during the night
- You can set the timer of feeds when do you want to feed the chicken or the fish

<h3> Cons: </h3>

- Need internet to function
- No electricity it won't function also
- Can only be controlled through the use of mobile phone
- The mobile application is applicable only to one prototype
